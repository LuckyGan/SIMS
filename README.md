Student Information Management System    学生信息管理系统

1、系统功能描述

	系统管理学生的课程信息，提供账户管理和学生信息管理两大功能。
	
	提供的功能有：
		
  	1）学生信息的查询、修改、增加、删除
		
 	2）账户信息的查询、修改、增加、删除
  
  	登录系统的账户分为管理员账户和普通用户。
		
  	1）普通用户只具有查询学生信息的功能
		
  	2）管理员用户具有系统的所有功能
  
  	主要的数据结构有：链表（存储账户和学生信息）
  
  	系统数据存储在文件中：
  
  	1）USER_ACCOUNT.txt  存放用户账户信息
		
  	2）USER_INFO.txt     存放学生信息
